# 🌿 NutriBot: AI Health & Diet Assistant

A terminal-based conversational agent that provides nutrition and wellness advice using state-of-the-art Large Language Models (LLMs) hosted on Hugging Face.

## 🚀 Features
- **Real-time Conversational AI**: Powered by Google's Gemma-2-2b-it model via the Hugging Face Inference API.
- **Context-Aware**: Remembers recent parts of the conversation for better advice.
- **Zero-Footprint**: Uses serverless inference—no local GPU or heavy model downloads required.

## 🛠️ Tech Stack
- **Language**: Python 3.9+
- **API**: [Hugging Face Inference API](https://huggingface.co/docs/api-inference/index)
- **Library**: `huggingface_hub`
